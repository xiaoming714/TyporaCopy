# markdown够用指南

[toc]



## 分级标题

#+空格表示几级标题

## 强调

用**或__包围

**这是两个*的强调**

__这是两个_的强调__

或者选中想要强调的文字按下Ctrl+b

## 斜体

用*或_包围

*这样*

_或者这样_

或者选中想要斜体的文字按下Ctrl+i

## 删除线

用~~包围

~~删除~~

~~看我看我都看我，我宣布个事，我是个伞兵~~

## *高亮(扩展语法)

==我喜欢黄色, 也喜欢绿色==

## 代码块

三个`+语言然后回车

```cpp
#include <iostream>
int main() {
    cout << "hello world" << endl;
    return 0;
}
```

## 引用

">"+空格

> 这是第一个引用
>
> >
> >
> >这是二级引用

## 无序列表

"-"或"+"+空格

- 这是一个无序列表
- 这还是一个无序列表

## 有序列表

数字+.+空格

1. 看我
2. 看我
3. 都看我
