# 数据结构

## 单调栈

对于单调递增栈，若栈顶元素被弹出，则当前元素为其右边第一个比它小的数。

对于单调递增栈，若当前元素被放入栈中，则栈中左侧的元素为第一个小于等于当前元素的数。

所以单调栈的作用在于求得一个元素在原始序列中左侧第一个大于他的元素，右侧第一个小于他的元素（当其被弹出时求得）。

一次单调递增栈的过程可以求得每个元素左侧第一个小于他的元素，右侧第一个大于他的元素。

## 二叉树

### 深度

求深度是可以递归求左右子节点中深度最大的，注意到达叶子节点时需要加一

## 堆

### 概念

如果有一个关键码的集合K = {k0，k1， k2，…，kn-1}，把它的所有元素按完全二叉树的顺序存储方式存储在一个一维数组中，并满足：Ki <= K2i+1 且 Ki<=K2i+2 ，则称为小堆(或大堆)。将根节点最大的堆叫做最大堆或大根堆，根节点最小的堆叫做最小堆或小根堆。

### 性质

堆中的节点总是不大于或不小于其父节点的值

堆始终是一颗完全二叉树
